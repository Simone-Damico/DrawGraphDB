Manager¶

Overview¶

Il manager gestisce i vari progetti creati dagli utenti ed è un classico project Django ed è quindi formato dai moduli tipici del framework.

Views¶

Le vievs ricevono dati dall’interfaccia, li elaborano e li usano per gestire i vari progetti.
